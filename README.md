## 📌 Sobre o projeto

O **SPRINT2-FRONT** é um projeto front-end desenvolvido em ambiente acadêmico com o objetivo de aplicar conceitos fundamentais de desenvolvimento web.

O foco principal é a construção de uma interface estática, organizada e funcional, utilizando boas práticas de estruturação com HTML, estilização com CSS e interatividade básica com JavaScript.

Este projeto simula uma aplicação web real, reforçando habilidades essenciais para o desenvolvimento front-end moderno.

## 🎯 Objetivos

- Desenvolver habilidades em **HTML5, CSS3 e JavaScript**
- Aplicar estruturação semântica e organizada
- Criar interfaces limpas e responsivas
- Praticar separação de responsabilidades no código
- Simular um ambiente real de desenvolvimento front-end

---

## 🛠️ Tecnologias utilizadas

- HTML5 → estruturação do conteúdo
- CSS3 → estilização e layout visual
- JavaScript → interatividade básica
